## 学习方法

- 使用 Playground 可以很好的练习语言
   - 安装 xcode
   - 安装 playgrounds
- 日积月累, 刻意练习
   - 苹果有一个说法, 结合 Playgrounds 的用法就是: REPL (Read - Eval - Print - Loop), 翻译为中文就是 "阅读 - 评估 - 打印(执行) - 循环" 这个方法非常适合阅读学习代码, 或者在编写一段代码之前的代码效果评估阶段, 多执行, 多分析, 可以更加扎实的巩固记忆.
